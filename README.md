# Photography Website

A responsive Photography Website built using HTML, CSS, and JavaScript.

---

## Features

- Animated Sidebar Navigation
- Dark / Light Mode
- Image Gallery Slider
- Upload Image Preview
- Contact Form
- Events Page
- Feedback Page
- Responsive Design
- Smooth Animations
- Auto Background Changer

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Pages

- Home Page
- Gallery Page
- Contact Page
- Events Page
- Feedback Page
- Shortcuts Page
